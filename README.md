# android-mvp-intent-sample

### Uma Intent no Android

Uma Intent é um objeto que representa uma intenção de ação dentro do sistema Android.
Ela é usada para:

abrir outra Activity

iniciar um Service

enviar um Broadcast

pedir que outro app execute algo. ```ex: abrir câmera, compartilhar texto```
